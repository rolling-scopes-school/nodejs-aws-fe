Serverless framework example app

This project was bootstrapped with Create React App and Serverless framework

Requirements

  • NodeJS 12+
  • AWS CLI
  • AWS account
  • AWS CLI must be configured with personal credentials

Setup

npm install -g serverless
npm install

Deployment

npm run cloudfront:update:build:deploy:nc
